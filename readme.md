# DiscordJS V13 Template

## Installation

### 1. Clone repository
```dos
git clone https://github.com/DanericNetwork/DiscordJS-Template.git
```
### 2. Setup config file
Rename `config.example.json` to `config.json`  
Fill all the fields

### 3. Install node packages
```dos
npm install
```

### 4. Run the bot
```dos
node index.js
```
